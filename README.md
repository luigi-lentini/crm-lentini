# CRM LENTINI

Complete project documentation.

## Overview
This project aims to develop a Customer Relationship Management (CRM) system for managing client and deal information.

## Features
- User authentication
- Client management
- Deal tracking
- Activity logging
- Reporting and dashboards

## Setup Instructions
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory.
3. Follow the setup guide in the `docs/SETUP.md` file for environment configurations.
