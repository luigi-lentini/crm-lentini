# CRM LENTINI Project

This repository contains the initial structure for the CRM LENTINI project that includes a Node.js backend, a React frontend, and a PostgreSQL database setup.

## Directory Structure

```
crm-lentini/
├── backend/
│   ├── server.js
│   └── package.json
├── frontend/
│   ├── src/
│   │   └── App.js
│   └── package.json
└── database/
    └── schema.sql
```
